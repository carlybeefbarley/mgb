All the tools you need

MyGameBuilder has all the tools you need within the site. This means all the students need is a browser and access to the internet to start using MyGameBuilder to start learning to code and building games without downloading anything. Along with our own code editor built-in with learning tools, students can also use our various asset editors to make graphics, sound, and music for their games, and share their creations with anyone.

An environment for cultivating learning

With MyGameBuilder students can access tutorials and various resouces to help with their learning journey. With a community of like-minded learners and game builders, students can work with each other and help each other out, or ask the mentors for help in the site. Help is within reach for any student at any time, so students can work at their own pace and ask for help whenever they need it.
